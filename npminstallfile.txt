passport (backend):
$ npm install @nestjs/passport passport passport-local @nestjs/jwt passport-jwt bcrypt
$ npm install -D @types/passport-local @types/passport-jwt

nestjs config (backend):
$ npm install --save @nestjs/config
$ npm install dotenv

typeorm, mysql(backend):
$ npm install --save @nestjs/typeorm typeorm mysql2

nestjs schedule (backend):
$ npm install @nestjs/schedule

mutler file upload (backend)
$ npm i -D @types/multer
$ npm install @nestjs/platform-express multer

bootstrap, bootstrap icons (front end):
$ npm install bootstrap
$ npm install bootstrap-icons

jwt (front end):
$ npm install @auth0/angular-jwt

sweetalert2 (front end):
$ npm install sweetalert2

testing (front end)

ng test
